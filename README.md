# 📊 E-Commerce Sales Dashboard

## 🔍 Project Overview
This project focuses on analyzing e-commerce sales data to identify trends, customer behavior, and product performance.

## 🎯 Objective
To transform raw data into meaningful insights for better business decision-making.

## 📈 Key KPIs
- Total Revenue  
- Total Orders  
- Profit Analysis  
- Total Quantity

## 💡 Key Insights
- Identified top-performing product categories contributing maximum revenue  
- Highlighted low-performing products requiring improvement strategies  
- Analyzed monthly and seasonal sales trends  
- Observed customer purchasing patterns across categories  

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning & Transformation  
- Interactive Dashboard (Slicers & Charts)  

## 📌 Outcome
This project helped in strengthening skills in data analysis, dashboard creation, and business insight generation.
